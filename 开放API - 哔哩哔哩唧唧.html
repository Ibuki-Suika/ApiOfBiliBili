通过视频AV地址获取视频(可直接获取福利视频CID，无需登陆)CID:

请求方式:GET


请求数据:

字段 取值 是否必需 描述
Av Int(数值类型) 是 bilibili视频的Av地址
P Int(数值类型) 否 bilibili视频的分P地址(如果单P可留空或者输入1)

请求地址:http://www.bilibilijj.com/Api/AvToCid/{Av}/{P}


请求示例:
http://www.bilibilijj.com/Api/AvToCid/10086
http://www.bilibilijj.com/Api/AvToCid/10086/1
http://www.bilibilijj.com/Api/AvToCid/10086/0 (当page为0时，将会返回整个集合)


返回内容格式:JSON


返回内容示例:{"code":0,"msg":"OK","av":10086,"p":1,"title":"【中文配音】搞笑漫画日和之贴纸","cid":1111282}


返回代码说明:

返回代码 说明
0操作成功完成
-1获取CID失败,比如视频不存在或者指定分P不存在;
-2参数错误，请严格按照上方请求地址格式化URL




通过CID获取音乐下载地址

请求方式:POST


请求数据:

字段 取值 是否必需 描述
CID Int(数值类型) 是 bilibili的视频CID
AV Int(数值类型) 是 bilibili的视频AV

请求地址:http://www.bilibilijj.com/ashx/mp3file.ashx


请求示例:cid=2345292&av=1542783


返回内容格式:JSON


返回内容示例:{"code":"1","mp3":"/Files/DownLoad/2345292.mp3","mp4":"/Files/DownLoad/2345292.mp4","mp4lenght":12668880,"mp3lenght":3639526}

PS:Code为状态代码,下面有详细状态说明


返回代码说明:

返回代码 说明
-5 上传时发生致命错误,无法完成上传指令！
-3 文件大小超出唧唧能承受的极限范围了！(300M)
-1,-2 解析出错，下载终止
0 队列中,队列名次可以通过读取"row"参数获取
1 OK
21 单段视频下载中...
22 分段视频下载中...
31 视频转码中...
41 提取MP3文件中...
51 正在上传MP4文件...
61 正在上传MP3文件...

其他参数:

参数 说明
mp3 MP3文件下载地址,在地址末尾加上"/www.bilibilijj.com.mp3"可以直接进行下载
mp4 mp4文件下载地址,在地址末尾加上"/www.bilibilijj.com.mp4"可以直接进行下载
mp3lenght mp3文件文件大小
mp4lenght mp4文件文件大小
row 当前在队列中排在第几位（仅当Code为0时才有此参数）


说明:虽然哔哩哔哩唧唧没有对API进行任何访问限制，不过还是希望您在使用API时注意请求频率。
